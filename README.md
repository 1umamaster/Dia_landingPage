# Dia — Strategic Agency

Dia is a modern, responsive landing page for a strategic creative agency. The project focuses on a bold visual identity, smooth user experience, and a clean layout to showcase the agency's expertise in branding, communication, and strategy.

## Technologies Used

* **HTML5** — Semantic markup for better SEO and accessibility.
* **SCSS/SASS** — Modular styles using BEM methodology and mixins for responsiveness.
* **JavaScript (ES6+)** — For interactive elements and logic.
* **Swiper.js** — To create a smooth, responsive hero slider.
* **BEM Methodology** — For clean, scalable, and maintainable CSS classes.

## Getting Started

Follow these instructions to set up the project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/1umamaster/Dia_landingPage.git
   cd Dia_landingPage

2. **Install dependencies:**
   ```bash
   npm install

3. **Run the project locally:**
   ```bash
   npm start

## Features

* **Responsive Design** — Fully optimized for mobile, tablet, and desktop views using media queries and custom mixins.
* **Interactive Hero Slider** — Custom navigation and smooth transitions powered by Swiper.js.
* **Smart Navigation** — Includes a functional off-canvas mobile menu and smooth scrolling for better UX.
* **Clean BEM Structure** — Modular SCSS architecture for easy maintenance and scalability.
* **Modern UI/UX** — Focus on bold typography, strategic whitespace, and professional aesthetics.
